### 2. Kaomoji encoding fixed (`app/api/chat/route.ts`)

18 garbled character sequences were replaced with the correct Unicode.  
The file is valid UTF-8 throughout вҖ” the issue was corrupted codepoints (e.g. `\u0080` control chars embedded in kaomoji), not a codec mismatch.  
Fixed with Python binary read/replace (`open(path, 'rb')` вҶ’ `.decode('utf-8')` вҶ’ string replace вҶ’ `.encode('utf-8')` вҶ’ `open(path, 'wb')`).

**Do NOT use the Edit tool for future kaomoji/garbling issues** вҖ” old_string byte matching will fail. Use Python.
